Overview

Don't waste time searching and enconding bank data anymore. Bank Autocomplete contains more than 100k pre-encoded banks that you can easily search and add.

Screenshots

Search among 100k+ banks

Quickly find the bank you are looking for. Odoo allows you to search bank by name or by BIC/SWIFT code

Pre-encoded data

No need to manually complete the bank's data. It's already done. All the encoded banks contain at least the bank's full name, the BIC/SWIFT code, the country and the city.

Installation

No specific manipulations required. Just add the app folder to your addons folder and install it from the Odoo client or from command line.
